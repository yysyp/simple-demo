#pslab/Kuaima.java is a thymeleaf based quick code generator.
#Mock usage:
1): http://localhost:8080/mock to create mock.
2): http://localhost:8080/mock/api/{YourMockUri} to call the mock.
On the "creating mock" page, the headers and body content will be parsed by thymeleaf engine.
and the "request" variable will be passed to the script context. So basically you can put the
content as below: {"requestMethod": "[(${request.method})]", "randomStr": "[(${#strings.randomAlphanumeric(8)})]"}

#Questionnaire poll Usage:
1): http://localhost:8080/api/qn/questionnaire  to create questionnaire form.
2): http://localhost:8080/api/qnpoll/{YourQuestionnaireUri}  to fill questionnaire form.
3): POST http://localhost:8080/api/qnpoll/{YourQuestionnaireUri}/result to submit questionnaire.
4): http://localhost:8080/api/qnpoll/{YourQuestionnaireUri}/result-excel to download questionnaire result excel (limits to 60000 rows).
5): http://localhost:8080/api/qn/questionnaire-result  view questionnaire result on page.
6): http://localhost:8080/bootcss-p-layoutit.html for html drag & drop creating HTML form snippet.
